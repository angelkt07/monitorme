MonitorMe is a customizable health diary app that was created for a quicker, neater way to keep track of loved ones, or your own health issues.

Some common questions about the app are:

What was your inspiration behind this application?

It was designed around a case study of a parent with a child that suffers from chronic migraines due to photophobia. The two were struggling with being able to keep up with a headache diary due to obstacles of everyday life. This application is meant to be a solution to that problem.

What are the icons in the menu bar?

Each button leads you to a specific place in the application. The list is as follows:

Home page

Summary & History page

References page

Settings page

Sign Out

How do I create a list of questions to monitor me?
Select the MonitorMe button on any page to get started. From here you can add the name, question and all answers. See example in the next section.
An example of a MonitorMe tracker!

First you need to name your list of questions, and then add your questions with the answers you want available to that question. It shows you three to start, but don't worry you can add more if necessary after saving the first three.

How do you edit questions and answers in a MonitorMe list that's already been created?

Click on the button, and on that page under summary click the tracker name in the first column. Then select the question or answer to go to the edit page. From there you can make changes!

How do I add more questions to a tracker that has already been created?

Simply click the icon in the menu bar at the top. You will then see a chart under the summary section. Select the title for the particular list in the first column that you want to add to and from there you have the option to add more questions and answers! See the next section to add more answers to a particular question.

How do I add more answers to question?

Click the icon in the menu bar and open the specific MonitorMe list to see the detail page. From there you have the current list of questions and answers. Underneath each set of answers, you have a link to add more answers to the specific question.
